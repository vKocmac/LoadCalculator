Psychrometrics Guide (input order)

  1. Run Load Calculation first, so room Qs/Ql exist.
  2. Select Supply mode: fixed Tsa or fixed RHsa%.
  3. Enter SA airflow only if known. If blank, it is estimated from Qs and Tsa.
  4. Use Coil model only when you want ADP/BF checks.
  5. Press Recalculate mix/coil and review Coil load, OA%, SA state.
